Once installed, add the extension to the page as you would with any other extension. Go to the page module, select a column and add a new content item. Then select Portfolio from from the list of content types.

    Configure the instance of Portfolio using the FlexForm

    NB: Use the 'startingpoint' field and the secondary options field 'recursive' to specify where records should be looked for.

    Display Mode

    The following options are available, they affect what is displayed on your website

    List

    1. Displays a list of portfolio items.

    Grid

    1. Displays a list of portfolio items in a grid (useful for just showing images of items).

    Single

    1. Displays detailed information about items.

    Client List
    Client

    1. Displays a list of clients.
