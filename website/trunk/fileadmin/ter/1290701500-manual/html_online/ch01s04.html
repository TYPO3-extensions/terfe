FAQ

  1. Possible subsections: FAQ

  2. OK (-1): These events indicate that everything went fine, no error occurred (at least up to that point where the event was created). No action needs to be taken.

  3. Info (0): These events are purely informational. They are normally used for debugging purposes only and require no special action.
